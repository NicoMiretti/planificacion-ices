# Glosario

| Término | Definición |
|---|---|
| **Planificación** | Documento Word elaborado por el profesor titular con la programación de su materia para un período académico. Tiene **campos obligatorios** definidos por la institución. |
| **Plantilla** | Documento Word base que el profesor debe completar. Existen **dos modelos**: uno para ICES y otro para UCSE. |
| **Componentes obligatorios** | Campos que deben estar presentes sí o sí: Propósito, Fundamentación, Contenidos mínimos, Metodología, Requisitos para regularizar y promocionar, Criterios y formatos de evaluación, Bibliografía. La ausencia de cualquiera de ellos provoca **rechazo automático**. |
| **Instancia de presentación** | Convocatoria abierta por la Secretaría Académica para que un grupo de profesores presente sus planificaciones. Tiene fecha de apertura, fecha límite, audiencia (carreras/institución) y período cubierto. |
| **Versión** | Cada envío de la planificación por el profesor. Numeración correlativa (v1, v2, …). |
| **Versión oficial** | Versión aprobada para un período. Única por (profesor, materia, período). |
| **Versión vigente** | Versión oficial actualmente en uso para el período en curso. Si se modifica una oficial, la nueva pasa a vigente y la anterior queda en histórico. |
| **Período académico** | Lapso al que corresponde la planificación: **año lectivo**, **1° cuatrimestre** o **2° cuatrimestre**. |
| **Materia anual** | Materia cuya planificación cubre todo el año lectivo. No requiere ratificación a mitad de año. |
| **Materia cuatrimestral** | Materia que se planifica por cuatrimestre. Si se dicta en ambos, se planifica por separado. |
| **Entrega tardía** | Envío realizado después de la fecha límite de la instancia. No se bloquea, pero se marca. Atrasos > 10–15 días se consideran significativos. |
| **Doble aprobación** | Resolución acordada entre la moderadora y el coordinador correspondiente. Ambos deben aprobar. |
| **Moderadora** | Secretaría Académica. Rol único que coordina la operatoria, recibe, valida formato y archiva. |
| **Coordinador** | Responsable académico por carrera/área. Aprueba en conjunto con la moderadora. |
| **Institución** | ICES o UCSE. Determina la plantilla aplicable y el grupo de notificación. |
| **Repositorio oficial** | Carpeta donde residen las planificaciones aprobadas (hoy: carpeta Drive 2). |
| **Carpeta de materiales** | Recursos para el profesor (modelo, reglamento, calendario, guía APA). Hoy: carpeta Drive 1. |
