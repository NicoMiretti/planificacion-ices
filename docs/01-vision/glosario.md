# Glosario

| Término | Definición |
|---|---|
| **Planificación** | Documento elaborado por el profesor con la programación de su materia (contenidos, actividades, evaluaciones, bibliografía, cronograma). Se entrega una vez por período académico. |
| **Versión** | Cada envío de la planificación realizado por el profesor. Se numeran correlativamente (v1, v2, …). |
| **Versión oficial** | Versión aprobada por el moderador para un período académico determinado. Solo puede haber **una** por (profesor, materia, período). |
| **Período académico** | Lapso al que corresponde la planificación: **año lectivo** (anual) o **cuatrimestre** (1° o 2°), según la materia. |
| **Materia anual** | Materia cuya planificación cubre todo el año lectivo. |
| **Materia cuatrimestral** | Materia cuya planificación cubre un cuatrimestre. Requiere una planificación por cuatrimestre dictado. |
| **Profesor** | Usuario que crea y envía planificaciones. |
| **Moderador** | Usuario que revisa y aprueba/rechaza planificaciones. (Posibles sinónimos: coordinador, jefe de cátedra — confirmar). |
| **Estado** | Situación actual de una versión: Borrador, Enviada, En revisión, Aprobada, Rechazada. |
| **Observación** | Comentario del moderador asociado a un rechazo o pedido de cambio. |
