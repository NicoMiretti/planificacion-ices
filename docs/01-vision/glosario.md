# Glosario

| Término | Definición |
|---|---|
| **Planificación** | Documento mensual elaborado por el profesor con la programación de su materia (contenidos, actividades, evaluaciones). |
| **Versión** | Cada envío de la planificación realizado por el profesor. Se numeran correlativamente (v1, v2, …). |
| **Versión oficial** | Versión aprobada por el moderador para un período determinado. Solo puede haber **una** por (profesor, materia, período). |
| **Período** | Mes calendario al que corresponde la planificación. |
| **Profesor** | Usuario que crea y envía planificaciones. |
| **Moderador** | Usuario que revisa y aprueba/rechaza planificaciones. (Posibles sinónimos: coordinador, jefe de cátedra — confirmar). |
| **Estado** | Situación actual de una versión: Borrador, Enviada, En revisión, Aprobada, Rechazada. |
| **Observación** | Comentario del moderador asociado a un rechazo o pedido de cambio. |
